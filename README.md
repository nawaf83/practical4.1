# SCC2026 Class Homepage

## 📖 About This Project

This repository contains the **Homepage** for the **SCC2026** class, developed as part of **Practical 4.1**. The project demonstrates fundamental web development skills using HTML.

## 🌐 Homepage Overview

The homepage (`Homepage.html`) is a simple HTML webpage for the SCC2026 class that includes:

- **Page Title:** "SCC class" — displayed in the browser tab
- **Welcome Message:** A paragraph introducing the class with the text *"This is my first class"*
- **Sponsor Link:** A hyperlink to [UTAR (Universiti Tunku Abdul Rahman)](http://www.utar.edu.my), acknowledging UTAR as the sponsor of the class

## 📁 File Structure

| File | Description |

| `Homepage.html` | The main homepage for the SCC2026 class |
